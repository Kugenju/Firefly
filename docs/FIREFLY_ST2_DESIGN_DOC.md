## 一、角色概述

### 背景故事
流萤是格拉默铁骑的幸存者，编号AR-26710。作为战胜虫群的基因改造战士而生，她驾驶着战术装甲「萨姆」战斗。为了寻找「生」的机会，她加入星核猎手，在宇宙中追寻违抗命运的方式。

> "火萤这种生物很神奇吧？它们或许会扑向火烛，或许会突然老化。但在那之前的每一个夜晚，它们的光芒会比星星更耀眼。"

## 二、核心机制

### 1. 灼热 (Scorch)
**机制描述**: 当带有灼热的敌人进行攻击意图时，立即受到对应层数的伤害，然后灼热层数减少1（每次攻击意图触发一次）。

**规则详情**:
- 触发时机: 敌人显示攻击意图时，在其行动前触发
- 伤害计算: 每层灼热造成1点伤害
- 衰减机制: 每次触发后层数-1
- 上限: 无上限，可无限叠加
- 视觉表现: 敌人身上带有火焰特效

**玩家效果**: 玩家打出攻击牌时，受到等于灼热层数的伤害，然后层数减少1。

---

### 2. 萤火 (Firefly) - 新增机制 ⭐
**机制描述**: 流萤卡组的核心属性机制，类似猎手的奇巧牌、战士的消耗机制。

**萤火状态**:
- **未激发**: 普通状态，正常效果，正常耗能
- **激发**: 卡牌效果翻倍，耗能-1（至少1点）

**触发条件**:
- 打出「完全燃烧」会激发所有手牌中的萤火牌
- 部分卡牌可以直接激发萤火牌

**视觉表现**:
- 未激发: 卡牌边框呈淡蓝色微光
- 激发: 卡牌边框燃烧绿色火焰，并有萤火虫粒子特效

**设计意图**:
- 萤火牌作为流萤的特色卡牌类型，不依赖完全燃烧也能发挥作用
- 激发机制提供额外的决策点：何时激发、如何配合
- 避免"没抽到完全燃烧就变成废物"的问题

---

### 3. 完全燃烧 (Complete Combustion) - 重做 ⭐
**机制描述**: 流萤的强化形态，改为以萤火机制为核心。

**效果详情**:
- 持续时间: 3回合
- **激发所有萤火牌**: 手牌中所有萤火牌进入激发状态（效果翻倍，耗能-1）
- 额外抽牌: 每回合多抽1张牌
- 伤害减免: 受到的所有伤害-5
- 退出条件: 持续回合结束自动退出

**与旧版对比**:
| 项目 | 旧版 | 新版 |
|------|------|------|
| 攻击力加成 | +3 | 移除 |
| 萤火效果 | 无 | 激发所有萤火牌 |
| 防御能力 | 无 | 受到伤害-5 |
| 抽牌 | 每回合+1 | 每回合+1（保留） |

**设计意图**:
- 与萤火机制深度绑定，提供爆发窗口
- 伤害减免替代攻击力加成，更注重生存能力
- 激发手牌中的萤火牌，提供即时战斗力提升

---

### 4. 生命消耗机制
**机制描述**: 部分卡牌消耗生命值而非/额外于能量。

**规则详情**:
- 生命值不足时，仍可打出（生命值降至1）
- 生命消耗不计入受到伤害（不触发受伤相关效果）

## 五、关键词定义

### 灼热 (Scorch)
- 带有灼热的敌人在显示攻击意图时，立即受到等于灼热层数的伤害，然后层数-1。
- 玩家打出攻击牌时，受到等于灼热层数的伤害，然后层数-1。
- 属于状态效果，可被清除。

### 萤火 (Firefly) ⭐新增
- 卡牌的固有属性，类似猎手的「奇巧」。
- **未激发**: 正常效果，正常耗能。
- **激发**: 效果翻倍，耗能-1（至少1点）。
- 可通过「完全燃烧」、卡牌效果等方式激发。

### 完全燃烧 (Complete Combustion) ⭐重做
- 持续3回合的强化状态。
- **激发所有萤火牌**: 手牌中所有萤火牌进入激发状态。
- 每回合多抽1张牌。
- 受到的所有伤害-5。

### 萤火护盾 (Flicker Shield)
- 抵消下一次受到的全部伤害。
- 持续1回合，触发后消失。

## 六、战斗循环设计

### 理想循环 - 萤火路线
1. **积累阶段**: 使用「萤火燎原」给关键牌添加萤火属性
2. **激发阶段**: 打出「点燃大海」或「完全燃烧」激发萤火牌
3. **爆发阶段**: 使用「海一直燃」打出所有激发状态的萤火牌
4. **收尾阶段**: 利用激发状态的高效果牌清理战场

### 理想循环 - 灼热路线（保留）
1. **叠加阶段**: 使用灼热卡牌叠加层数
2. **控制阶段**: 利用灼热伤害在敌人行动前削弱
3. **爆发阶段**: 「完全燃烧」提供生存能力，继续叠加
4. **收割阶段**: 敌人行动时触发大量灼热伤害

### 双路线融合
- 萤火牌也可以施加灼热（如火萤斩击）
- 「完全燃烧」同时服务于两条路线
- 玩家可以根据抽牌灵活切换策略

## 七、代码实现要点

### 萤火关键词定义
```csharp
public class FireflyKeywords
{
    [CustomEnum("FIREFLY")]
    [KeywordProperties(AutoKeywordPosition.Before)]
    public static CardKeyword Firefly;
}
```

### 萤火状态管理
```csharp
public class FireflyCardModifier : CardModifier
{
    public bool IsIgnited { get; set; }
    
    public override void Apply(CardModel card)
    {
        if (IsIgnited)
        {
            // 效果翻倍
            // 耗能-1
        }
    }
}
```

### 完全燃烧新效果
```csharp
public class CompleteCombustionPower : PowerModel
{
    public override void OnTurnStart()
    {
        // 激发所有手牌中的萤火牌
        foreach (var card in Owner.HandPile.Cards)
        {
            if (card.HasKeyword(FireflyKeywords.Firefly))
            {
                card.GetModifier<FireflyCardModifier>().IsIgnited = true;
            }
        }
        // 额外抽牌
        Owner.DrawCards(1);
    }
    
    public override int ModifyDamageReceived(int damage)
    {
        return damage - 5; // 伤害减免
    }
}
```
